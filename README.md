# A-High-End-Refrigerator-Display-Panel
Team project for designing a high-end refrigerator with a user display panel
## Project Vision
For people who want a refrigerator that’s simple to use and gives helpful information right on the screen Who need real‑time updates about their food, temperature settings, and any issues with the appliance our product is a smart refrigerator display system built in Java that shows what food you have, lets you adjust temperatures, sends maintenance alerts, and connects with smart‑home devices Unlike basic refrigerators our product gives users more control, clearer information, and energy‑efficient features.
